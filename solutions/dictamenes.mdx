Un **dictamen profesional** de Continuum Lands es el punto de partida para tomar decisiones informadas sobre cualquier predio en México. A través de un análisis técnico exhaustivo, evaluamos el uso de suelo, la zonificación, los coeficientes de densidad, la altura máxima permitida, la proximidad a servicios y el valor estimado de mercado — todo consolidado en un reporte estructurado que elimina la incertidumbre antes de invertir, desarrollar o asesorar.

<Note>
  Cada dictamen cruza **más de 320 variables** de fuentes oficiales, registros catastrales, datos de movilidad urbana y métricas de mercado en tiempo real para ofrecerte el análisis más completo disponible.
</Note>

## ¿Qué incluye un dictamen?

Un dictamen cubre las dimensiones técnicas, normativas y de valor que determinan el potencial real de un predio. A continuación se describen los campos principales que encontrarás en cada reporte:

| Campo                  | Descripción                                                             |
| ---------------------- | ----------------------------------------------------------------------- |
| **Superficie**         | Área total del predio en m²                                             |
| **Frente**             | Longitud del frente del lote en metros lineales                         |
| **Zona**               | Clasificación administrativa y delegación/alcaldía                      |
| **Uso de suelo**       | Clave normativa (p. ej. H3/30/M — habitacional, 3 niveles, 30% libre)  |
| **Valor/m²**           | Estimación del valor de mercado por metro cuadrado                      |
| **COS**                | Coeficiente de Ocupación del Suelo (porcentaje de la planta construible) |
| **CUS**                | Coeficiente de Utilización del Suelo (metros cuadrados totales construibles respecto al lote) |
| **Altura máxima**      | Número de niveles permitidos por la normativa vigente                   |
| **Distancia a metro**  | Tiempo/distancia caminando a la estación de metro más cercana           |
| **Riesgo sísmico**     | Clasificación de zona según el Atlas de Riesgo (Zona I, II o III)       |
| **Walkability score**  | Índice de caminabilidad de 0 a 100 basado en servicios y conectividad   |

## Ejemplo real: Av. Insurgentes Sur 1234, Benito Juárez, CDMX

Para ilustrar el nivel de detalle de un dictamen, el siguiente ejemplo corresponde a un predio de **320 m²** ubicado en Av. Insurgentes Sur 1234, alcaldía Benito Juárez, Ciudad de México:

| Parámetro           | Valor obtenido          |
| ------------------- | ----------------------- |
| Superficie          | 320 m²                  |
| Valor/m²            | $48,200 MXN             |
| COS                 | 0.6                     |
| CUS                 | 1.8                     |
| Altura máxima       | 6 niveles               |
| Walkability score   | 82 / 100                |
| Riesgo sísmico      | Medio (Zona II)         |

### Puntuación por categorías

Además de los datos normativos, el dictamen asigna puntuaciones interpretativas para facilitar la comparación entre predios:

| Categoría                  | Puntuación  |
| -------------------------- | ----------- |
| Servicios y comercio       | 86 / 100    |
| Conectividad               | 78 / 100    |
| Plusvalía estimada         | 72 / 100    |

## ¿Quién se beneficia de un dictamen?

Un dictamen profesional de Continuum Lands es útil en distintos perfiles del ecosistema inmobiliario:

- **Inversionistas** que evalúan una adquisición y necesitan confirmar el valor real y el potencial del predio antes de cerrar una operación.
- **Desarrolladores** que requieren verificar la edificabilidad (COS, CUS, altura) antes de diseñar un proyecto arquitectónico.
- **Asesores y agentes inmobiliarios** que necesitan fundamentar sus recomendaciones con datos técnicos y argumentos de valor para sus clientes.
